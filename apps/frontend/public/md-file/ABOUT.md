## 项目概述

> 本项目为社区内容打造智能聚合浏览平台，专注于提升内容发现与阅读体验。

#### 功能

- **搜索**: 采用向量化搜索引擎构建索引，支持精准匹配与多维度筛选。
- **个性化**: 用户可设定自己的性取向，灵活关注或屏蔽作品，关键词，作者，频道等全部层级。
- **浏览**: 提供关注流、热门、分区等个性化视图，并能方便的管理自己关注的作者和收藏夹。
- **其他**: 平台集成Discord OAuth验证及自定义样式

#### 筛选

- **排序**: 4种排序方案可自行选择，并有综合发布时间、互动权重的智能算法。
- **视图**: 提供卡片，列表，极简视图，提供滚动和翻页分页方式。
- **作品**: 作品详情页在网页端已经渲染，可以快速跳转到其中任何链接，最终无缝跳转至社区详情页。
- **作者**: 为创作者提供了个人主页，聚合自己的数据和作品，并快速索引跳转功能。

## 技术栈

- **前端**: Vue 3 + Vite + Naive UI
- **后端**: Fastify + Kysely + Redis
- **认证**: Discord OAuth
- **搜索引擎**: Meilisearch
- **数据库**: PostgreSQL
